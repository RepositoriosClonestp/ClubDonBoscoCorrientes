# Assets del Club Don Bosco

## Logo

Colocar aquí el archivo `logo.png` con el logo oficial del Club Don Bosco. 

**Especificaciones recomendadas:**
- Formato: PNG con fondo transparente
- Tamaño: 500x500 px mínimo
- Relación de aspecto: 1:1 (cuadrado)

El logo se utilizará en: 
- Encabezado de la aplicación (menú lateral)
- Recibos PDF
- Reportes exportados
